# TypeWeather

TypeWeather é um aplicativo mobile simples de previsão do tempo onde o usuário podera pesquisar uma cidade brasileira e o app retornara a ele informações climaticas de previsão dessa região
com um tema personalizado dependendo do clima de tal.

## Páginas

### Home

-Página inicial onde o usuario pesquisara em um input dinámico que retorna recomendações de cidades de acordo com oque foi pesquisado

### Result

-Página que traz o resultado definitivo da busca do usuário com um tema personalizado e as seguintes informações:
- Situação das nuvens (dispersas, sem nuvens, nublado e etc)
- Temperatura máxima e mínima
- Sensação termica
- Probabilidade de chuva
- Velocidade do vento
- Umidade do ar
- Índice UV
- Previsão para os proximos 5 dias

  Tambem possue um input para que o usuario faça uma outra busca por essa página.

### User

-Página de visualização das informações do perfil, onde o usuário podera atualizar e ver suas informações,são elas, foto de perfil, nome, email e senha.

### CreateMovie

-Página para criação de avaliação com inputs de nome, categoria, desrição, estrelas e etc.

## Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/Toddynh0BR/TypeWeather.git
```

Acesse o diretório do projeto:

```bash
cd TypeWeather
```
Instale as dependências(recomendo firmemente que use yarn):
```bash
yarn add
```

Inicie o servidor de desenvolvimento:
```bash
yarn start
```

ou

```bash
yarn android
```

ou

```bash
yarn ios
```

ou

```bash
yarn web
```

Sinta-se à vontade para contribuir com o projeto.
